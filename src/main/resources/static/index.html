公司介绍

        呦呦龙珠北京国际科技教育有限公司团队建立于2010年，致力于研发IQ、EQ双商课程，为中国儿童和家庭提供高品质多元化的国际化教育。
        公司旗下汇集了FUTURE ENVISION、金智立、龙笃笃等众多优秀教育品牌，拥有强大的研发及教学团队，培养了近万名在教育领域成绩斐然的教师；和美国、加拿大、新加坡等著名国际院校建立合作关系。教育核心体系将智商、情商课程完美融合，系统化、专业化、国际化、人性化。呦呦教育填补国内培训机构课程单一的空缺，用爱、责任、专业的教育理念，打造世界一流教育王国。为中国家庭提供专业的线上、线下一站式教育服务、定制家庭教育方案，让中国儿童从优质的早期教育中受益，拥有学习能力和探索世界的好奇心。
        呦呦教育将进一步承担起推动儿童教育进步的重任，不断致力于教育资源的均衡全方位发展，实现从教到育、从学到习、从身到心的全面提升。

团队介绍

韩超

龙笃笃课程研发主力成员

媒体、教育、互联与公益领域跨界创新者。曾经通过Harvard SEED社会创新summer camp在哈佛肯尼迪政治学院学习领导力课程。历任国际在线主持、主编、网络电台总监，家庭阅读跟踪研究者，儿童阅读分级推广者，积极推动提升儿童公共表达能力。

李杰

清华大学AI博士

现旅居美国。人工智能、数据挖掘、语义分析专家，在大众心理学、教育学领域均有涉猎，尤其关注中美教育体系的比较研究和相互作用，以及华裔儿童、青少年的社会和情绪能力培养。主管龙笃笃理论技术架构。

北京大学经济学学士

长期从事传文化、传播、培训等工作，致力于儿童情商培养与阅读能力养成体系研究多年，著有包括龙笃笃系列在内的儿童作品数十种，并主导研发了龙笃笃儿童情商课程。主管龙笃笃产品研发。

李红薇

资深插画师
